## BOM
| Component | Part | Quantity |
|---|---|---|
| Diodes | 1N4148 | 4 |
| RGB LEDs | SK6812MINI | 6 |
| Display | OLED 128x32 | 1 |
| Switches | MX Push Switch | 3 |
| Encoder | Rotary Encoder | 1 |
| MCU | Seeed XIAO RP2040 | 1 |

## Firmware
Built with QMK. Features:
- Desktop switching shortcuts
- Volume control via encoder
- RGB volume indicator
- OLED display
